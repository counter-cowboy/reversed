Напиши метод, который принимает на вход строку и меняет порядок букв в каждом слове на обратный:
- с сохранением регистра (например, 'Cat' -> 'Tac', 'houSe' -> 'esuOh', 'elEpHant' -> 'tnAhPele' - регистр сохраняется в соответствии с позицией прописных и строчных букв в первоначальном слове);
- с сохранением пунктуации (например, 'cat,' -> 'tac,', "is 'cold' now" -> "si 'dloc' won" - знаки препинания остаются на своих местах).

Обрати внимание: составные слова, содержащие дефис(ы) и/или апостроф(ы),  считаются отдельными словами с символами-разделителями между ними (например, 'third-part' -> 'driht-trap', 'can`t' -> 'nac`t').

Также напиши unit-тесты для этого метода.
